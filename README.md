# Medicine Inventory Management System

A Python-based console application to manage pharmacy inventory, including adding, updating, deleting, and selling medicines.

## üöÄ Features

- **Add Medicine:** Enter details like name, batch number, company, price, quantity, expiry date.
- **Update Stock:** Modify the quantity of an existing medicine.
- **Order Medicine:** Check stock, calculate bill, update inventory after sale.
- **Delete Record:** Remove medicine using batch number.
- **View Inventory:** Display all records in a structured table.
